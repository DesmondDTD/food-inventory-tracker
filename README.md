# Desmond's Food Inventory Tracker

A full-stack web application built with Flask, SQLite, and Bootstrap that helps users easily track, categorize, and monitor their food (or general inventory) — including expiration dates, quantities, and categories.  
It features **user authentication**, a visual **dashboard**, and a clean, modern UI.

## Features

**User Authentication** — Register, log in, and manage your own inventory securely.  
**Add / Edit / Delete Items** — Track item name, quantity, category, and (optional) expiration date.  
**Expiration Alerts** — Items nearing expiration are automatically highlighted.  
**Dashboard Analytics** — Visualize your inventory by category using Plotly.  
**Responsive UI** — Mobile-friendly design using Bootstrap 5.  
**Optional Expiration Dates** — Works for food, office supplies, or any item type.  

## Tech Stack

- Backend: Flask, Flask-Login, Flask-Bcrypt, Flask-SQLAlchemy  
- Frontend: HTML, CSS, Bootstrap 5, Plotly.js  
- Database: SQLite  
- Language: Python 3
